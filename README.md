# Portofolio Website

Website portofolio pribadi modern dan responsive yang dibuat menggunakan HTML, CSS, dan JavaScript. Project ini menampilkan profil, skill, pengalaman, pendidikan, serta project yang pernah dikerjakan.

## Preview

Website ini memiliki tampilan modern dengan desain clean dan responsive sehingga dapat diakses dengan baik melalui desktop maupun mobile.

## Fitur Utama

* Responsive Design
* Modern UI/UX
* Animasi Fade In
* Navigation Menu Mobile
* Section About, Skills, Projects, Experience, Education, dan Contact
* Tampilan portfolio profesional
* Mudah dikustomisasi

## Teknologi yang Digunakan

* HTML5
* CSS3
* JavaScript
* Font Awesome
* Google Fonts (Poppins)
